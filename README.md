# 📝 Simple Python To-Do List

A lightweight, web-based To-Do List application built with **Streamlit**. This app allows you to manage your daily tasks efficiently, with all data persisted locally in a **CSV** file.

## 🏢 Built with:
   - [Python](https://docs.python.org/3/)
   - [Streamlit](https://streamlit.io/)


## 🚀 Features
- **Add Tasks:** Quickly type in new items.
- **View List:** See all your current tasks in a clean table.
- **Persistent Storage:** Tasks are saved to **CSV** files so they remain even after closing the app.
- **Complete/Delete:** Easily remove tasks once they are finished.
